Esta aplicacion funciona con Streamlit, puede instalarlo usando el comando: pip install streamlit
Puede verificar la instalacion corriendo el siguiente comando: python -m streamlit --version
para correr una instancia localhost del entorno grafico de la aplicacion en su navegador use el comando: python -m streamlit run main.py
